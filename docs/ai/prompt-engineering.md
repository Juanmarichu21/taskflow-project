# Prompt Engineering aplicado al desarrollo

## Objetivo

En este documento se recopilan prompts utilizados para programación y desarrollo del proyecto TaskFlow.

---

# Prompt 1

## Prompt
"Actúa como un desarrollador senior de JavaScript y refactoriza esta función"

## Resultado
La IA simplificó la lógica y mejoró nombres de variables.

## Conclusión
Definir un rol mejora la calidad técnica de la respuesta.

---

# Prompt 2

## Prompt
"Explica paso a paso cómo funciona esta función"

## Resultado
La IA explicó el flujo completo del código.

## Conclusión
Pedir razonamiento paso a paso ayuda a comprender mejor el código.

---

# Prompt 3

# Conclusión final

El prompt engineering permite obtener mejores resultados al dar instrucciones más específicas.

Las técnicas más útiles fueron:
- definir roles
- pedir razonamiento paso a paso
- usar restricciones
- proporcionar ejemplos

Cuanto más claro es el prompt, mejor es la respuesta generada por la IA.
